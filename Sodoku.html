Sodoku Solver

Here is the Sodoku solving algorithm that I created. It utilizes recursive backtracking in order to solve these puzzles. The sodoku grid being worked on is represented by a 3d array. The program progresses through the grid, each call of solveRB() working on a singular box. The program will attempt to put a number into the box that it is working on until one works, then it will move onto the next box. If the program is able to complete the entire grid with a working solution then the Sodoku has been solved. Othewrwise when numbers don't work, the program will backtrack to the last box it has worked on and continue to try numbers.
