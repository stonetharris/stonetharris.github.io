Blackjack Game

Here are all of the classes that I wrote in order to create the Blackjack game.

First is the simple Card class. This class includes all the functionality that a card requires in order to play any card game: suit and rank identifications, two different constructors, and getter methods.

  • Card.java

Next is the Deck class. Built to be able to handle and manipulate arrays consisting of cards, it has shuffling and dealing functionality.

  • Deck.java

Then is the Player is class. This class gives the player an entity that can store a hand. Functionality for adding cards, hand evaluation (what the hand's value is and whether they have Blackjack), and discarding cards is included with this class. Additionally, this class allows the player to inpit whether or not they want a hit by using scanner.

  • Player.java

Extending from the Player class is the Dealer class. This class utilizes the base of the Player class, but overrides two main methods: the printHand method, because the player is not supposed to be able to see the dealer's hand and the wantsHit method, because the dealer wanting a hit can be determined by the value of his hand.

  • Dealer.java

The class that ties everything together is the Blackjack class. Its methods do the heavy lifting in terms of carrying out the game. It utilizes methods such as playRound and printResult in order to proceed through rounds, determining whether the player or the dealer wins each round.

  • Blackjack.java
